Data Fields

bool alwaysVisible = false
 If true, the UI will be active at all times. Use this for scenes dedicated to remapping where you never want the UI hidden.
bool enableKeyInput = true
 If true, the toggle key can be used to show/hide the UI. You can turn this off to use gesture-based or event-only toggling.
Key toggleKey = Key.F1
 The keyboard key used to toggle the UI visibility. This will only be used if enableKeyInput is true.
GameObject uiRoot
 The root GameObject for the UI to show or hide. This GameObject will be enabled/disabled during toggling.
UnityEvent OnUIToggle
 Optional UnityEvent that can trigger a toggle externally (e.g., from a Lemming gesture). External scripts or the inspector can invoke this at runtime.

Detailed Description

Controls the visibility of the remapping UI at runtime. Supports toggling via keyboard input or UnityEvents (e.g., gesture-based triggers).
